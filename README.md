# Yi-Ting's Page

一個簡單的個人網頁範例，展示現代化 Glassmorphism 設計，並動態顯示目前時間。

---

## Demo 連結

👉 [線上示範頁面](https://eating-thinker.github.io/DIC1-personal-page-with-antigravity-and-github/) — 這是這個專案已部署的 GitHub Pages 網站，可以直接點開查看實際效果。{index=0}

---

## 功能特色

- **動態時間顯示**：每秒更新使用者當前時間。  
- **現代化設計**：
  - 線性漸層背景
  - 玻璃風格卡片（半透明 + 模糊效果）
  - 懸停時卡片有輕微浮動動畫
- **自適應排版**：內容置中，適應各種螢幕尺寸

---

## 技術細節

- **HTML5**：結構化標記  
- **CSS3**：
  - Flexbox 用於置中
  - 線性漸層背景與玻璃效果
  - 文字陰影與過渡效果
- **JavaScript**：
  - 使用 `Date` 物件取得當前時間
  - `setInterval` 每秒更新時間
  - 24 小時制顯示

---

## 使用方式

1. 將 `index.html` 儲存到本地  
2. 用瀏覽器開啟即可  
3. 修改 HTML 中的文字（如 `Hello, Yi-Ting!`）來客製化問候語

---

## 開發建議

- 可改為 12 小時制並加上 AM/PM  
- 可以加入更多互動元素，例如按鈕或連結  
- 調整背景漸層或透明度以適應不同主題
